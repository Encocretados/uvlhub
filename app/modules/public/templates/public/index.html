Welcome to UVLHub.io!

What is uvlhub.io?

UVLHub.io is a repository for feature models in UVL (Universal Variability Language) format, created by DiversoLab to support research in software variability. Integrated with Zenodo for open-access sharing and FlamaPy for model analysis, UVLHub.io allows users to upload, cite, and explore feature models in a standardized format, fostering collaborative research and advancing studies in software product lines.

Can I contribute?

Sure! You can make a fork of our official repository on GitHub, create fantastic features and then make a pull request to us.
